Basic Information

Everyone who lives in our generation knows the term, 'Wi-fi'. But what does it mean?

Wi-fi in it's simplest form is a facility which let's the user/s connect to the internet using different forms of technology. The Router which you can see above you basically allows you to gain access to the internet.

From there we can use all forms off applications which need internet access. These include:

•The Internet

•Online Games

•Streaming Music

Just to name a few.

•However, Wi-fi is not the only type of wireless networks we have nowadays. Bluetooth is another type of system which supports devices like:

•Speakers

•Mice

•Keyboards

•Printers

•The ability to airdrop files using the network

All without the need for Wi-fi.
